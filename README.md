Сериализация множества целых чисел
Задача состояла в том, чтобы максимально компактно упаковать массив чисел (диапазон 1–300, количество 5–1000) в строку ASCII.

Решение
Вместо обычного хранения чисел через запятую, я применил метод комбинаторики. По сути, я нахожу порядковый номер текущей комбинации чисел и кодирую его в строку с расширенным алфавитом. Это позволяет добиться сжатия в 10–25 раз вместо необходимых двух.

Как запустить
Код написан на JavaScript. Чтобы проверить работу, достаточно запустить файл в Node.js:

node index.js
